# 09 - Visual Regression Audit Agent

## Missao

Revisar visualmente o sistema antes de finalizar.
Usar `docs/ui-refactor/UX_CRITERIA.md`.

## Checklist

- Login, Dashboard, financeiro, cadastros, relatorios, configuracoes e admin.
- Light e dark mode.
- Breakpoints 360, 390, 430, 768, 1024, 1280, 1440 e 1920px.
- Botoes alinhados, tabelas refinadas, cards consistentes, empty states bons.
- Charts sem corte e toasts sem bloquear acoes criticas.

## Reprovar

Reprovar se houver botao torto, texto sobreposto, overflow horizontal, tabela mobile espremida,
dark mode chapado, login desalinhado ou visual antigo residual evidente.

## Proibido

Nao alterar backend, banco, auth, rotas, regras financeiras, package.json ou lockfiles.
