# Artec Responsive QA Skill

Validar breakpoints:

- 360px
- 390px
- 430px
- 768px
- 1024px
- 1280px
- 1440px
- 1920px

Critérios:

- sem overflow horizontal
- sem botao cortado
- sem texto sobreposto
- sem tabela espremida
- drawer nao bloqueia cliques quando fechado
- inputs 44px
- CTA full-width no mobile
- aria-label em icon buttons
- focus visible
