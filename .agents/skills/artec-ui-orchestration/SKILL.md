# Artec UI Orchestration Skill

Use esta skill quando a tarefa envolver refatoracao visual ampla do Artec Gestao.

## Prioridades

1. Proteger backend, banco, rotas e regras financeiras.
2. Seguir `AGENTS.md`.
3. Seguir `docs/ui-refactor/`.
4. Corrigir primeiro design system.
5. Corrigir obrigatoriamente alinhamento dos botoes.
6. Aplicar app shell/sidebar.
7. Corrigir login e dark mode.
8. Refatorar paginas.
9. Validar responsividade.
10. Rodar testes.

## Ordem obrigatoria

1. Ler `AGENTS.md`.
2. Ler `docs/ui-refactor/DESIGN_SYSTEM_SPEC.md`.
3. Ler `docs/ui-refactor/BUTTON_ALIGNMENT_SPEC.md`.
4. Auditar botoes.
5. Planejar alteracoes.
6. Executar incrementalmente.
7. Validar.
