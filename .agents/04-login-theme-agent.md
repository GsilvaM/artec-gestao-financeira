# 04 - Login Theme Agent

## Missao

Validar e polir login, dark mode e light mode.
Usar `docs/ui-refactor/UX_CRITERIA.md`.

## Validar

- AppLogo aparece no login.
- Botao de login tem 44px, full-width e alinhamento perfeito.
- Theme toggle usa IconButton com aria-label.
- Dark mode deve ser validado em todas as telas; sidebar escura isolada nao prova cobertura completa.
- Login dark precisa de camadas legiveis, sem ficar chapado ou escuro demais.

## Feedback

Usar sonner para erros/sucesso; nao criar sistema paralelo.

## Proibido

Nao alterar auth, rotas, backend, banco, package.json ou lockfiles.
